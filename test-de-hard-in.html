Absatz 27.0

Moderne Primzahltests für allgemeine Zahlen n können in zwei Hauptklassen, probabilistische (oder `` Monte Carlo'') und deterministische Algorithmen eingeteilt werden. Deterministische Algorithmen bieten eine Möglichkeit, um sicher zu sagen, ob eine bestimmte Zahl Prime ist oder nicht. Beispielsweise ist Trial Division ein deterministischer Algorithmus, da sie, wenn korrekt ausgeführt, immer eine Primzahl als Primzahl und eine zusammengesetzte Zahl als zusammengesetzt identifiziert. Diese Tests stützen sich auf das Testen einer bestimmten Zahl in einer teilweise zufälligen Weise. Beispielsweise kann ein Test die ganze Zeit bestehen, wenn er auf eine Primzahl angewandt wird, aber nur mit der Wahrscheinlichkeit p, wenn er auf eine zusammengesetzte Zahl angewandt wird. Wenn wir den Test wiederholen n mal und jedes Mal passieren , dann ist die Wahrscheinlichkeit, dass unsere Zahl zusammengesetzt ist 1/ ( 1-p ) n , die exponentiell nimmt mit der Anzahl der Tests , so können wir so sicher sein, wie wir wollen ( wenn auch nie ganz sicher ) dass die Zahl Primzahl ist . Auf der anderen Seite, wenn der Test jemals scheitert, dann wissen wir, dass die Zahl zusammengesetzt ist.
